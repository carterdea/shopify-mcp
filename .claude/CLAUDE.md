# Shopify MCP

## Tooling

- Use `bun` instead of `npm` or `node`
- Run `bun x ultracite fix` before committing

## Code Style

Follow Biome's rules (enforced via Ultracite). The linter handles formatting, imports, and common pitfalls automatically.

## Commands

```bash
bun x ultracite fix    # Format and auto-fix
bun x ultracite check  # Check for issues
bun test               # Run tests
```
